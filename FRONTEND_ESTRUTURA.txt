🚀 QUICK START

1. Instalar dependências:
   cd frontend && npm install

2. Configurar .env:
   cp .env.example .env
   # Editar se necessário (padrão está ok)

3. Iniciar dev server:
   npm run dev

4. Abrir no navegador:
   http://localhost:5173

🎯 FLUXO DE USUÁRIO

Novo Usuário
    ↓ Clica em "Registre-se"
    ↓ Preenche email, nome, senha
    ↓ ✅ Conta criada
    ↓ Não logado (redireciona para login)
    ↓
Login
    ↓ Email + password
    ↓ ✅ Token armazenado
    ↓
Home
    ↓ Ver pets disponíveis
    ↓ Clicar em "Tenho Interesse!"
    ↓ Modal: preencher nome + telefone
    ↓ ✅ Formulário enviado
    ↓
Dashboard (se admin)
    ↓ Abas: Pets | Interessados | Organizações
    ├─ Criar novo pet
    ├─ Ver interessados por pet
    ├─ Criar organização
    └─ Gerenciar organizações
